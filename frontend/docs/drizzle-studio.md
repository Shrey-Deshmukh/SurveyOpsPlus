# How to run drizzle studio

1. Run

```bash
npm install #to install plugin dep
```

1. Start dev server

```bash
npn start
```

1. Open in android of ios app using the menu

2. Reload app using 'r' and wait for the terminal to re-install the application

3. In the same terminal press

```txt
shift + m # or mac equivalent to open dev tools
```

1. Go down and select expo-drizzle-studio-plugin

These steps should open up a browser tab with an interactive MYSQL studio like UI
